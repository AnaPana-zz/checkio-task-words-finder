You are given a text and string with key words (or parts of words), separated by spaces. You need to find all key words, ignoring upper or lower cases and put these words into "" wrappers for highlighting. All letters cases should be kept.

Input: Two arguments. Both are strings

Output: String with replaced key words.

Example: 

assert(checkio("This is only a text example for task example.", "example") == 
        "This is only a text example for task example."), "Simple test"
assert(checkio("Python is a widely used general-purpose, high-level programming language.", "PyThoN") ==
        "Python is a widely used general-purpose, high-level programming language."), "Ignore letters cases"
assert(checkio("It is desirable to replicate a randomized experiment as closely as possible by obtaining treated and control groups with similar covariate distributions.", "is im") ==
        "It is desirable to replicate a randomized experiment as closely as possible by obtaining treated and control groups with similar covariate distributions."), "Several subwords"

How it is used: Searching machines.
